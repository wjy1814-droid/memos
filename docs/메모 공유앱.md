# 메모 공유 앱

## 1. 프로젝트 개요

### 1,1 프로젝트명
메모 공유 앱

### 1.2 프로젝트 목적
사용자간 메모를 공유 하는 앱

### 1.3 개발 범위
- 메모 작성 및 작성한 메모 확인 화면
- 사용자간 메모 내용을 공유
- 메모를 생성/조회/수정/삭제할 수 있는 기능

## 2. 기술 스택
- 프로트엔드: HTML, CSS, 리엑트, 자바스크립트
- 백엔드: Node.js, Express
- 데이터베이스: PostgreSQL

## 3. 기본사항
- 프론트엔드와 백엔드를 따로 개발
- 기본적인 웹 기술만 사용
- 학습 목적이므로 사용자 인증이나 경제 기능 제외
- 기본 하늘 바탕에 메모는 구름형태에 글자는 검정색
